About WasmTS

JTS Topology Suite 1.20.0 compiled to WebAssembly using GraalVM Native Image (26 EA) with web-image backend. Direct Java-JavaScript bindings via @JS annotations. Runs entirely client-side.

Features
  • Geometry types: Point, LineString, Polygon, Multi*, GeometryCollection
  • Operations: buffer, union, intersection, difference, convexHull, simplify
  • Spatial predicates: contains, intersects, touches, crosses, within, overlaps, covers, equalsTopo
  • Coordinate support: 2D, 3D (XYZ), and 4D (XYZM)
  • STRtree spatial indexing, PreparedGeometry
  • LineMerger, CascadedPolygonUnion, OffsetCurveBuilder, Densifier, GeometryFixer
  • WKT, WKB, and GeoJSON input/output

Visualization uses backproj + maplibre-proj to display geometry in EPSG:2249 (MA State Plane) centered on Boston City Hall, via proj-wasm for in-browser coordinate transformation.
